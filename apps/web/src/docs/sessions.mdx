# Sessions & Chat

Learn how to create sessions and interact with your characters.

## Creating a Session

A session brings together a character and setting for interaction.

### Steps

1. Click **Sessions** in the sidebar
2. Click **New Session**
3. Select a character from your library
4. Select a setting from your library
5. Click **Start Session**

## The Chat Interface

Once in a session, you'll see:

- **Chat history** - All messages in the conversation
- **Input box** - Where you type your messages
- **Character info** - Name and basic details
- **Setting info** - Current location context

### Sending Messages

1. Type your message in the input box
2. Press `Enter` or click **Send**
3. Wait for the character to respond

## How NPCs Process Input

Behind the scenes, your message goes through multiple agents:

1. **Intent Detection** - What are you trying to do?
2. **Sensory Processing** - What does the character perceive?
3. **Emotional Analysis** - How does this affect their mood?
4. **Knowledge Retrieval** - What relevant info do they recall?
5. **Response Generation** - Crafting the reply
6. **State Update** - Updating character memory

## Tips for Great Interactions

### Be Descriptive

❌ "I walk in"
✅ "I push open the heavy wooden door and step into the tavern, shaking snow from my cloak"

### Ask Open Questions

❌ "Do you have food?"
✅ "What's on the menu tonight? I've been traveling for days."

### Reference Context

Use details from:

- The setting description
- Previous conversation
- Character background

### Build Relationships

- Remember what characters tell you
- Reference past conversations
- Show how your actions affect them

## Session Management

### Continuing Sessions

- Sessions are automatically saved
- Resume from the Sessions library
- All conversation history is preserved

### Multiple Sessions

You can have:

- Different sessions with the same character in different settings
- Multiple ongoing conversations
- Parallel storylines

## Advanced Features

### Debug Panel

Enable the debug sidebar to see:

- Agent routing decisions
- Intent detection results
- Context retrieval details
- Response generation timing

### Configuration

Customize session behavior:

- LLM model selection
- Temperature settings
- Context window size

## Troubleshooting

**Character not responding?**

- Check your internet connection
- Verify the API server is running
- Look at the debug panel for errors

**Responses feel generic?**

- Add more detail to your messages
- Reference specific character traits
- Mention setting elements

**Character forgets context?**

- This is expected for very long conversations
- Important info should be in character/setting descriptions
- Consider starting a new session for fresh topics
