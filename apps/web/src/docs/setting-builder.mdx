# Setting Builder

Create immersive worlds and locations for your characters to inhabit.

## Overview

Settings in ArcAgentic provide the environmental context for character interactions. A well-designed setting includes atmosphere, locations, and world-specific knowledge.

## Key Components

### Basic Information

- **Name** - The identifier for your setting
- **Description** - What makes this place unique?
- **Type** - World, region, location, or specific area

### Atmosphere & Mood

Describe the sensory experience:

- **Visual** - Colors, lighting, architecture
- **Auditory** - Background sounds, music
- **Emotional** - How does this place make people feel?

### Location Hierarchy

Build nested locations:

```
World: Eldoria
  └─ Region: Northern Territories
      └─ Location: Frostpeak Village
          └─ Area: The Rusty Anvil Tavern
```

## Creating a Setting

1. Navigate to **Settings** in the sidebar
2. Click **Create New Setting**
3. Fill in the basic information
4. Add atmospheric details
5. Define any special rules or characteristics
6. Save your setting

## Example Settings

### The Moonlit Tavern

```
Description: A warm refuge on a lonely crossroads
Atmosphere: Crackling fire, smell of roasted meat, hushed conversations
Notable Features: Ancient map on the wall, mysterious innkeeper
```

### The Crystal Mines

```
Description: Deep underground tunnels lit by glowing crystals
Atmosphere: Echo of pickaxes, cool damp air, eerie blue light
Notable Features: Unstable passages, valuable ore deposits, ancient ruins
```

## Best Practices

- **Be specific** - "Warm firelight" beats "nice lighting"
- **Engage multiple senses** - Sight, sound, smell, touch
- **Consider implications** - How does the setting affect character behavior?
- **Leave room for imagination** - Don't over-specify every detail

## Integrating with Characters

When starting a session:

- Characters will reference the setting in their responses
- Environmental details influence NPC behavior
- Setting knowledge affects what characters know
